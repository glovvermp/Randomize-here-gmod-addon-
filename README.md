# Аддон на гмод
### "Randomize here!"

Моя версия мода для созадания точек с рандомным спавном заданных предметов.

> [!CAUTION]
> прототип версия! максимально сырая и имеет скудный функционал, а также может иметь баги и недочеты!

> [!NOTE]
> Предметы для ввода указывать в данном формате:
> ```
> weapon_pistol, weapon_smg1, weapon_shotgun
> ```
> А затем нажмите "применить", тогда выбранные предметы начнут случайно появляться во всех существующих точках.
