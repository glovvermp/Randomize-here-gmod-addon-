# Аддон на Garry's mod
### "Randomize here! – Spawn Point Randomizer"

Моя версия аддона для созадания точек с рандомным спавном заданных предметов.

Первый аддон написаный мною. Glua изучался в процессе создания и весь аддон был написан примерно за неделю. Планируются обновления и поддержка.

> [!CAUTION]
> **Возможны баги.** Мод не полностью проверен и будет патчится в процессе.
>> **У точек нет лимита! Осторожнее с их количеством на карте.**
>
>> **Взаимодействовать со всем функционалом на данный момент могут все игроки. Вскоре отлючу часть/все функции для игроков не имеющих права администратора.**

> [!NOTE]
> Предметы для ввода указывать в меню мода. Следует создать ID предмета и нажать на имя в списке. Откроется меню настроки данного ID где можно указать желаемые предметы их шанс появления и количество _(количество предметов определяется в случайном диапазоне от 1 до вашего заданного количества)_
>
> А затем нажмите "применить/Apply", и ID точек появятся в списке в меню инструмента.

### Установка 
Скопируйте или переместите папку "spawn_tool" в папку игры:
GarrysMod\garrysmod\addons

Для лицензии в steam (если вам не известно как установить):
1. Откройте Garry's mod в библиотеке
2. Нажмите на гайку (справой стороны от "играть") управление -> посмотреть локальные файлы
3. Откройте папку garrysmod и там будет папка addons 


> [!NOTE]
> Зачастую мод установленный в папку игры не отображается в меню "Дополнения" внутри игры. Это нормально, ведь в данном меню на лицензионной версии игры отображаются лишь предметы из мастерской на которые вы подписанны.

### Комманда для бинда на определенную кнопку

Вы можете активировать все существующие точки на карте через нажатие кнопки клавиатуры, а не в меню инструмента

откройте игровую консоль (нажатием клавиши ~) и введите к данную комманду:

```
 bind "N" "RH_activate_all"
```

Вы можете установить удобную вам клавишу. Просто замените букву внутри первых кавычек. к примеру _"G" "ctrl" "f1" или "MOUSE4" для боковой кнопки мыши_
